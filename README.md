# Kitchen & Lifestyle Study

A brand-neutral, mobile-first customer discovery survey built with Next.js, TypeScript, and Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

Survey: `/`  
Private local results: `/admin`

## Important data note

This initial prototype stores progress and completed responses in the visitor's browser. Responses from a shared public link will therefore **not** aggregate on the researcher's device. Connect a hosted database or form endpoint before using it for distributed research.
